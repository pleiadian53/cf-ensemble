    üöÄ Development Workflow¶

    Recommended approach (as per project organization):

    1. Develop example script under examples/<topic>/
    2. Pure Python, executable with argparse
    3. Import from src/cfensemble/

    4. Save outputs to results/<topic>/

    5. Test thoroughly

    6. Unit tests in tests/

    7. Integration test via script execution

    8. Create notebook (optional, for pedagogy)

    9. under notebooks/<topic>/
    10. Import from example script

    11. Add narrative and visualizations

    12. Document under docs/methods/<topic>/

    13. Theoretical background
    14. API documentation
    15. Link to examples and notebooks
