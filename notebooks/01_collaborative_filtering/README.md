# Collaborative Filtering with ALS

This directory contains notebooks introducing collaborative filtering concepts applied to ensemble learning.

## Notebooks

- **Demo-Part1-CF_with_ALS.ipynb**: Introduction to optimization via Alternating Least Squares (ALS) for CF-based ensemble learning

## Key Concepts

- Base models as "users" and data points as "items"
- Rating matrix factorization
- ALS optimization for latent factor discovery
- Ensemble transformation through CF
