# Probability Filtering

This directory contains notebooks on CF ensemble learning with probability filtering and sequence models.

## Notebooks

- **Demo-Part5-CF_Ensemble_via_Probability_Filtering.ipynb**: Introduction to probability filtering in CF ensemble
- **Demo-Part5a-Alternative_Data_Representations_for_Probability_Filtering.ipynb**: Alternative data representations
- **Demo-Part5b-Probability_Filtering_via_Custom_Loss.ipynb**: Probability filtering with custom loss functions

## Key Concepts

- Probability filtering for ensemble refinement
- Sequence models for ensemble predictions
- Custom loss functions for probability estimation
- Alternative data representations
